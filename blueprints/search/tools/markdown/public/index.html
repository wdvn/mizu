How it works

Every URL goes through three tiers, falling back automatically until clean Markdown is produced.

Tier 1 · Native
Content negotiation — sites that natively serve text/markdown return it directly, with zero processing.
Tier 2 · AI
HTML fetched and converted via Cloudflare Workers AI — fast, structure-aware HTML-to-Markdown extraction.
Tier 3 · Browser
JS-heavy SPAs rendered in a headless browser before AI conversion — handles React, Vue, any SPA.

Responses are edge-cached for 1 hour with stale-while-revalidate. CORS enabled on all endpoints.
