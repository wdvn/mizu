native

quick start

Works from any language or shell.

Append any URL to the endpoint. POST for JSON with metadata.

# get markdown from any URL
$ curl https://markdown.go-mizu.workers.dev/https://example.com

# or POST for JSON with metadata
$ curl -X POST https://markdown.go-mizu.workers.dev/convert \
    -H 'Content-Type: application/json' \
    -d '{"url":"https://example.com"}'

faq

Common questions

Why Markdown?

HTML has navigation, ads, scripts, styling your agent doesn't need. Markdown keeps headings, paragraphs, links. ~80% fewer tokens.

Is it free?

Yes. Open access. No account, no API key. Use the URL directly.

JS-heavy pages?

Falls back to headless browser rendering. React, Vue, Angular, any SPA handled automatically.

Browser CORS?

All endpoints return Access-Control-Allow-Origin: *. Call from client JS directly.

Protected pages?

Only public URLs. Pages behind auth or paywalls return an error.

Limits?

5 MB max. 10s timeout, 20s for browser rendering. No hard rate limit.
